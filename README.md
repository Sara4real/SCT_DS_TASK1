📊 Task 1: Population Data Analysis (World Bank Dataset)
📝 Objective
Analyze global population data to visualize the distribution and identify top populated countries in the year 2022.

📁 Dataset
World Bank – Total Population Dataset

File: API_SP.POP.TOTL_DS2_en_csv_v2_2590.csv

Source: World Bank Data

🔢 Features Used
Country Name

2022 – Population value for the year 2022

🧪 What I Did
Cleaned the dataset and converted the 2022 population column to numeric

Removed rows with missing values

Sorted the countries by population

Visualized:

🔹 Top 10 countries by population (bar chart)

🔹 Population distribution across all countries (histogram)

💻 Tech Stack
Python

Pandas

Matplotlib

Seaborn

Jupyter Notebook

📈 Visualizations
Bar Chart – Top 10 most populated countries in 2022

Histogram – Distribution of population across all countries

🔍 Key Insights
India and China lead global population in 2022

Majority of countries fall within the lower-to-mid population range

A small number of countries hold the majority of the world’s population

🚀 How to Run
Clone the repository

Open the Jupyter Notebook task1_population_analysis.ipynb

Run the notebook step-by-step to generate visualizations
